# Scripts

## Installation

```bash
$ git clone https://github.com/yuyahnd/scripts.git
```

### bash
`~/.bashrc`
```bash title="~/.bashrc"
if [ -f ~/to-path/scripts/bash/.bashrc ]; then
    source ~/to-path/scripts/bash/.bashrc
fi
```

### zsh
`~/.zshrc`
```zsh title="~/.zshrc"
if [ -f ~/to-path/scripts/zsh/.zshrc ]; then
    source ~/to-path/scripts/zsh/.zshrc
fi
```
